# random-integer

> Generate a random integer

## Install

```
$ npm install --save @glowyjs/random-integer
```

## Usage

```js
const randomInteger = require('@glowyjs/random-integer');

randomInteger();
//=> 1109494507128900

randomInteger(0, 10);
//=> 5
```

## Tests

Run tests

```
npm run test
```
